# Reef animation plan

所有 row 保持相同帽型、臉型、虎紋、體格、短褲、背鰭與尾鰭。動作用身體重心、肢體和相連器材表達海上航行；透明背景不承載場景。

1. `idle`（6）：海腳站姿、呼吸、眨眼、極小頭部與尾鰭動作。
2. `running-right`（8）：面向畫面右側，抱緊繩圈，交替短步。
3. `running-left`（8）：獨立生成左向動作，保留虎紋與尾鰭方向一致性。
4. `waving`（4）：手觸帽簷敬禮，轉為揮手，再收回。
5. `jumping`（5）：屈膝預備、垂直跳起、頂點、下降、站穩。
6. `failed`（8）：雙手轉舵、用力、船舵卡住、低頭洩氣。
7. `waiting`（6）：抱著繩圈等候、看向使用者、略開掌詢問。
8. `running`（6）：站定轉舵，雙手交替，身體隨想像中的浪勢小幅傾斜。
9. `review`（6）：望遠鏡貼眼巡視，停頓後稍微放低。
10. rows 9–10：16 向視線；腳底和下半身固定，眼睛先行，頭頸與帽子跟隨，尾鰭只作極小反向平衡。
