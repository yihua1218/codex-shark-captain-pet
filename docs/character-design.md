# Reef character design

Reef 是成熟、精壯而友善的虎鯊獸人船長。他的氣質是經歷過風浪後的沉著，不是兇狠或滑稽；笑容能露出鯊魚牙齒，但眼神保持可靠、略帶調皮。

## Identity lock

- 白色寬頂船長帽，深藍帽簷、黃銅色帽帶；不放可讀文字或商標。
- 主體為中深藍色皮膚，深海軍藍虎紋，胸腹及口鼻為淺冰藍。
- 寬肩、厚胸、明顯但不過度零碎的胸腹肌；四肢結實、手腳比例適合小尺寸 sprite。
- 背部保留一片明確背鰭；尾巴由腰後連出，末端為鯊魚尾鰭，避免伸出格外。
- 深海軍藍甲板短褲，腰側保留少量紅藍格紋作為參考圖連結。
- 深色粗細一致輪廓，cel-shaded 高光；避免細小寫實皮膚紋理。

## Connected props

船舵、繩圈、望遠鏡只在對應狀態出現，而且必須接觸雙手、臉部或軀幹，形成單一可抽取的 sprite 元件。畫面不出現船身、海面、天空、甲板、陰影、浪花或漂浮圖示。
